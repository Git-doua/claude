# PL/SQL Code Repository

This repository contains PL/SQL database objects and scripts.

## Directory Structure

- **procedures/** - Stored procedures
- **functions/** - User-defined functions
- **packages/** - Package specifications and bodies
- **triggers/** - Database triggers
- **views/** - Database views
- **ddl/** - Data Definition Language scripts (CREATE TABLE, ALTER, etc.)
- **scripts/** - Utility and deployment scripts

## Usage

Place your PL/SQL files in the appropriate directories based on their type.

## Naming Convention

Consider using descriptive names for your SQL files, for example:
- `create_user_proc.sql`
- `get_employee_func.sql`
- `employee_pkg.sql`
